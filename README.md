# SkinCheckAI

SkinCheckAI adalah sebuah website berbasis kecerdasan buatan (Artificial Intelligence) yang dirancang untuk membantu pengguna dalam mendeteksi dan memprediksi jenis penyakit kulit secara otomatis menggunakan teknologi Deep Learning, khususnya Convolutional Neural Network (CNN).

Melalui platform ini, pengguna dapat mengunggah foto area kulit yang terindikasi bermasalah, kemudian sistem akan menganalisis citra tersebut dan memprediksi kemungkinan jenis penyakit kulit seperti dermatitis, jerawat, psoriasis, atau melanoma, lengkap dengan tingkat kepercayaan (confidence level) model.

Website ini dikembangkan dengan kombinasi antara frontend interaktif yang mengintegrasikan model deep learning terlatih.
